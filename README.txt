** MittensEdit **

Dependencies: Codemirror 6, Saitamaar.
Dev dependencies: esbuild

Build with node build.js

This should bundle the Codemirror dependencies in dist/codemirror.bundle.js

https://github.com/SikigamiHNQ/OrinrinEditor

rectangleselect.js incorporates code licensed under the MIT license from Codemirror 6.


For MittensEdit to function, download the Saitamaar font files and place them in ./font
Saitamaar can be downloaded here: 
https://github.com/asciiart-development/SaitamaarFont
https://fonts.aahub.org/font/6


Template info:
aalist.txt and aabrush.txt store single-line templates and brushes respectively. Both can be copied over from the ones found in （´д｀）Edit or OrinrinEditor.
For multi-line templates, place your MLT files (with ending .mlt) in templates/mlt and add the filenames to mltlist.txt
